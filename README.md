# mass-effect-codex

Turns the Mass Effect codex (trilogy + Andromeda) and its in-universe timeline
into a single entity co-occurrence graph, viewable as a self-contained
interactive HTML explorer.

**[Open the live explorer](https://cheremnyh-yu.github.io/mass-effect-codex-explorer/codex_explorer.html)**

Or download `codex_explorer.html` from this repo and open it directly in a browser.
